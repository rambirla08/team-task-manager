Team Task Manager

Overview:
Team Task Manager is a full-stack MERN application that helps teams manage projects and tasks efficiently.

Features:
- User Registration & Login
- JWT Authentication
- Create Projects
- Create Tasks
- Task Status Management
- Responsive UI
- Full-stack MERN architecture

Tech Stack:

Frontend:
- React.js
- Tailwind CSS
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- JWT Authentication
